Библиотечная система управления

Для запуска:
1. Установить MySQL и Python
2. Импортировать все library_db в phpMyAdmin
3. Установить библиотеку: pip install mysql-connector-python
4. Запустить: python library_app.py

Логин/пароль MySQL: library_user / library_pass123
(Логин и пароль можно посмотреть в коде Python "library_app.py")
